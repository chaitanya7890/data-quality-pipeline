
## Technologies Used

- Python  
- Pandas  
- Great Expectations  
- Git & GitHub  

## Pipeline Steps

1. Load raw dataset from CSV  
2. Validate dataset for missing values and invalid records  
3. Remove invalid rows and clean data  
4. Save cleaned dataset for downstream analytics  

## How to Run

Activate virtual environment
venv\Scripts\activate


Install dependencies
pip install -r requirements.txt


Run validation
python pipeline/validate_data.py


Run transformation
python pipeline/transform_data.py
